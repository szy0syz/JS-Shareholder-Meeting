# JS-Shareholder-Meeting

一个股东会辅助软件

## 主要功能

- 股东数据录入

- 股东签到

- 股东表决结果录入

- 制作表决结果

## 技术栈

- v1: Express.js + Bootstrap + Pug + MongoDB
- v2: Express.js + React + Redux + React Router + MongoDB
- v3: Express.js + Vue + Vuex + Vue Router + MongoDO
- v4: Koa.js + React + Redux + React Router + JWT + MongoDB
